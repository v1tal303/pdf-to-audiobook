# pdf-to-audiobook

<code>Pdf-to-audiobookr</code> is quick and simple solition to convert PDFs into audiobooks.

## How to use

1) Provide a location for the .pdf file

2) Run the script

3) Wait for the .mp3 files to appear in your folder
